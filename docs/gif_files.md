# GIF File Creation
The GIFs used in the experiment guide screen were created using Canva:

1. Go to <https://www.canva.com>.
2. Create a new design (card or custom dimensions as needed).
3. Add elements, animations, or screenshots to demonstrate the experiment steps.
4. Export the design as a GIF (Download → File Type → GIF).
5. lace the GIF file under `assets/images` and reference it in the experiment
guide screens.
